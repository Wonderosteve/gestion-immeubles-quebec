# Application Gestion Immeubles (Version GitHub Pro) Application simple pour téléphone pour gérer : - immeubles - logements - locataires - loyers - réparations - historique Copiez ce fichier dans **index.html** dans votre dépôt GitHub. ```html Gestion Immeubles

Gestion Immeubles

Résumé

Revenus: 0$

Dépenses: 0$

Profit: 0$

Ajouter locataire

Paiement de loyer

Nouvelle réparation

``` ## Déployer sur GitHub 1. Aller sur **github.com** 2. Cliquer **New repository** 3. Nom : `gestion-immeubles` 4. Cliquer **Add file → Upload files** 5. Envoyer **index.html** 6. Activer **Settings → Pages → Deploy from branch → main** Votre application sera disponible comme site web. Exemple : ``` https://votrenom.github.io/gestion-immeubles ``` Elle fonctionnera aussi comme **application mobile** si vous l'ajoutez à l'écran d'accueil. --- Améliorations possibles : - loyers en retard - photos de réparations - gestion de plusieurs immeubles - export pour impôts Québec - graphiques revenus
